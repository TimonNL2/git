Menu en gebruikersinvoer

In een console-programma kun je een menu maken door de gebruiker een prompt te geven met input(). Bijvoorbeeld: je toont opties en laat de gebruiker kiezen. De gebruiker typt een getal of tekst in en met if keuze == "1": … ga je door naar de bijbehorende actie.

Voorbeeld Menu-code:
print("1: Optie A") print("2: Optie B") keuze = input("Kies een optie: ") # Gebruiker voert in als "1" of "2" if keuze == "1": print("Je hebt Optie A gekozen") elif keuze == "2": print("Je hebt Optie B gekozen") else: print("Ongeldige keuze")

Hier pauzeert input() het programma en wacht op invoer. Door deze code in een while-lus te plaatsen, kun je het menu herhalen totdat de gebruiker kiest om te stoppen.

Variabelen voor gegevensopslag

Een variabele is een naam die naar een waarde wijst; je gebruikt variabelen om data te bewaren voor later gebruik. Je kent een variabele een waarde toe met het =-teken.

Voorbeeld Variabelen-code:
mijnGetal = 13 mijnTekst = "Hallo" print(mijnGetal, mijnTekst) # Drukt: 13 Hallo

In dit voorbeeld sla je het getal 13 op in mijnGetal en de tekst "Hallo" in mijnTekst. Variabelen kunnen getallen, tekst of ander soort gegevens bevatten en zijn flexibel te wijzigen.

Lussen (herhaling met while en for)

Een lus herhaalt een stuk code meerdere keren. Er zijn twee hoofdtypen: de while-lus en de for-lus.

while-lus

Een while-lus draait zo lang een bepaalde voorwaarde True is.

Voorbeeld while-code:
x = 5 while x > 0: # Herhaal zolang x groter is dan 0 print(x) # 5, 4, 3, 2, 1 x -= 1 # Verklein x met 1 print("Klaar!")

for-lus

Een for-lus is handig om over een reeks of lijst te itereren, bijvoorbeeld met range().

Voorbeeld for-code:
for i in range(1, 4): # i gaat achtereenvolgens 1, 2, 3 bevatten print("Dit is regel", i)

Dit drukt drie regels af. Een for-lus stopt automatisch na de laatste waarde in de reeks.

Functies schrijven en gebruiken

Met functies kun je code organiseren in herbruikbare onderdelen. Een functie definieer je met def en optioneel een return-waarde.

Voorbeeld Functies-code:
def groet(naam): return f"Hallo, {naam}" print(groet("Mark")) # Roep de functie aan met argument "Mark"

De functie groet neemt één parameter naam en geeft een groettekst terug. Zo houd je code overzichtelijk en repeteer je dezelfde logica.

CSV-bestanden lezen en bewerken

Python heeft de ingebouwde csv-module om gemakkelijk met CSV-bestanden te werken.

Lezen van CSV

Voorbeeld CSV Lees-code:
import csv with open('gegevens.csv', newline='') as csvfile: lezer = csv.reader(csvfile) for rij in lezer: print(rij) # Elke rij als lijst, bijv. ['Naam','Leeftijd']

Schrijven naar CSV

Voorbeeld CSV Schrijf-code:
import csv with open('gegevens.csv', 'w', newline='') as csvfile: schrijver = csv.writer(csvfile) schrijver.writerow(['Naam', 'Leeftijd']) schrijver.writerow(['Anna', '23'])

Om gegevens toe te voegen, open je in append-modus ('a') en gebruik je writer.writerow(). Om te bewerken, lees je alle regels in, wijzig je de rij in de lijst en schrijf je alles opnieuw weg.

Voorbeeldproject: Studentenbeheer

Hieronder zie je een volledig voorbeeld waarin alle bovenstaande elementen samenwerken in één programma: een menu, variabelen, lussen, functies en CSV-bewerking.

import csv FILENAME = 'students.csv' def read_students(): students = [] with open(FILENAME, newline='') as csvfile: lezer = csv.reader(csvfile) for rij in lezer: students.append(rij) return students def toon_studenten(): print("\nHuidige studenten:") for rij in read_students(): print(f"ID {rij[0]}: {rij[1]} (score {rij[2]})") def voeg_toe(): naam = input("Naam van de student: ") score = input("Score van de student: ") students = read_students() nieuw_id = len(students) # Aangenomen dat eerste ID 0 is; of +1 voor ID 1 etc. with open(FILENAME, 'a', newline='') as csvfile: schrijver = csv.writer(csvfile) schrijver.writerow([nieuw_id, naam, score]) print("Student toegevoegd.") def update_score(): sid = input("Geef het ID van de student die je wilt bijwerken: ") students = read_students() gevonden = False for i, rij in enumerate(students): if rij[0] == sid: nieuwe_score = input(f"Nieuwe score voor {rij[1]}: ") students[i][2] = nieuwe_score gevonden = True break if gevonden: with open(FILENAME, 'w', newline='') as csvfile: schrijver = csv.writer(csvfile) for rij in students: schrijver.writerow(rij) print("Score bijgewerkt.") else: print("Student niet gevonden.") def main(): while True: print("\nMenu:") print("1: Toon alle studenten") print("2: Voeg student toe") print("3: Update score student") print("4: Stoppen") keuze = input("Maak een keuze: ") if keuze == "1": toon_studenten() elif keuze == "2": voeg_toe() elif keuze == "3": update_score() elif keuze == "4": print("Programma beëindigd.") break else: print("Ongeldige keuze, probeer opnieuw.") # Start het programma main()

Dit programma toont een menu, leest en schrijft students.csv en laat je studenten toevoegen of bijwerken met behulp van functies, variabelen en lussen.
